# 网站性能优化 - 执行步骤

## ✅ 已完成的修改

1. **添加了图片压缩脚本** (`compress-images.js`)
2. **创建了懒加载组件** (`src/components/LazyImage.tsx`)
3. **更新了 Portfolio 页面** 使用懒加载
4. **添加了 sharp 依赖** 用于图片压缩

## 📝 执行步骤

### 步骤 1: 安装依赖
```bash
npm install
```

### 步骤 2: 压缩图片（重要！）
```bash
npm run compress
```
这会将 112MB 的图片压缩到约 20-30MB，节省 70-80% 的体积。

### 步骤 3: 测试效果
```bash
npm run dev
```
访问 http://localhost:3000/portfolio 查看效果

### 步骤 4: 构建部署
```bash
npm run build
```

## 🎯 优化效果

- **图片懒加载**: 只加载可视区域的图片
- **图片压缩**: 减少 70-80% 的图片体积
- **加载提示**: 显示加载动画，提升用户体验

## ⚠️ 注意事项

- 压缩是不可逆的，建议先备份原图
- 压缩过程可能需要 5-10 分钟
- 压缩后重新部署网站即可生效
